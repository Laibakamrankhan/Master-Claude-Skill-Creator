# IT Blog Writer Skill

This skill enables Claude to generate high-quality, professional IT blog articles that meet industry standards for technical accuracy, structure, and presentation.

## Overview

The IT Blog Writer transforms Claude into a senior software engineer and technical writer combined, capable of producing publish-ready technical content that stands out from generic AI-generated articles.

## Features

- **Technical Accuracy**: Ensures all information is factual and verifiable
- **Professional Structure**: Follows best practices for technical blog organization
- **Visual Enhancement**: Includes diagrams and code examples where appropriate
- **Audience Adaptation**: Adjusts complexity based on target audience
- **Modern Best Practices**: Focuses on current industry standards

## Usage

To use this skill, provide:

1. **Required**: Blog topic
2. **Optional**:
   - Target audience (beginners, intermediate, experts)
   - Preferred tech stack (React, Python, AWS, etc.)
   - Tone preference (educational, conversational, formal)
   - Depth/length (overview, detailed, comprehensive)

## Example Command

```
Use the IT Blog Writer skill to create an article about "Microservices Architecture Patterns" for intermediate developers using Docker and Kubernetes with an educational tone.
```
